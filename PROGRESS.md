# Running Animal - 戦国ドット絵テーマ 進捗

## プロジェクト概要
フラッピーアニマルの続編・拡張版を、戦国時代をモチーフにしたドット絵テーマで構築する。
HTML5 Canvas を使用したサイドスクロール型ランニングゲーム。

## テーマ: 戦国ドット絵
- **プレイヤー**: 金の豹（runner/フォルダのスプライト画像を使用）
- **背景**: 戦国時代の風景（城、山、桜、鳥居）をドット絵で描画
- **障害物**: 刀、手裏剣、槍、矢など戦国風の武器をドット絵で描画
- **演出**: パララックススクロール、ドット絵フォント、和風カラーパレット

## ファイル構成
- `index.html` - メインHTML（スタイリング含む）
- `game.js` - ゲームロジック全体
- `runner/` - 走る動物のアニメーションフレーム（3x3 = 9枚）

## 実装機能
- [x] ドット絵風の戦国背景（城、山、桜、雲）
